A foundation release. Every layer of the Sui-native private-send stack got scaffolding, one flow works end-to-end (brando's SUIAMI privatization), and the infrastructure bug that blocked the whole Walrus write path from the Cloudflare Worker is gone.

Named Gaius for the Roman namesake — a foundation name on which an empire built.

## End-to-end working

### Walrus Quilt publish from the Worker (Aggron Stone Edge)

First real blob landed: `snsTHkOcIFRWBBbtDG5Oc9M3aJRi-wN2buYsQ4ZcyjM`. Cascade of fixes needed:

- Transport: `SuiJsonRpcClient` instead of `SuiGraphQLClient` (5000B query cap)
- WASM: pre-init `walrus-wasm` via static `CompiledWasm` import (CF Workers disallow runtime `WebAssembly.instantiate()`)
- Epochs + deletable required on `writeQuilt`
- Upload relay at `upload-relay.mainnet.walrus.space` (storage-node fan-out unreachable from Worker)

Ultron now pays WAL + SUI to batch blobs into Quilts for any producer in the stack.

### SUIAMI privatization — first real user completion

`waap.sui` is fully Seal-gated:
- `verified: True`
- walrus blob: `wlT6Csc0L7m32p80L5vYIdNdDatR8A6KqtSnpvUFl4Y`
- seal nonce: set
- 2 dWallet caps (secp256k1 + ed25519)
- public chains: `[sui]` — btc/eth/sol/tron all live privately in the Walrus blob

First time any user completed the full `rumble → Seal encrypt → Walrus → on-chain roster → Seal-decryptable cross-chain addresses` loop.

### Gateway stealth mode on `addr(whelm.eth, 784)` (Beldum Double Hit)

Personal-mode identities now get a FRESH Sui stealth address per CCIP-read query. Service-mode identities (ultron, plankton) keep the stable address.

- Fluidkey pattern, gateway-side ECDH
- Uses `@noble/curves/ed25519` for ephemeral gen + `@noble/hashes/blake2` for Sui address encoding
- Recipient scanner picks up inbound via ephemeralPub channel (console-logged today, announcement channel is next move)

### squids CLI + keystore

`bun run squids list / add / get / publish / verify / chainAt / who`. Ultron + plankton SUIAMI'd (ultron has all 4 chains; plankton is sui-only pending rumble).

### Porygon Iron Tail — pre-deploy regression net

`bun run check:move` scans every `${pkg}::<mod>::<fn>` target in the TS source and dry-runs against mainnet. Catches the exact bug class that hit `upgradeSuiami` today — stale package pointer to a function that doesn't exist at the published-at address.

First live run caught a sibling of the bug it was built to prevent (`seal_approve_cf_history` targeting the original pkg instead of LATEST). Fixed before it fired.

### Paymaster signer persisted

Ultron's existing secp256k1 squid now persisted to `UltronSigningAgent` DO state via `POST /api/admin/paymaster-signer`. No code paste; rotation is a click.

## Scaffolded (landable but not yet end-to-end)

### Sneasel Shadow Ball pt1

Sui private-send port. `SneaselWatcher._sweepSui` branch dispatches per `chain` on tick. IKA ed25519 signing stubbed as `pending-ika-sign`; Icy Wind pt2 wires the live ceremony.

### UltronEnvelope v1 scaffold

Types + builders + validation for `transfer | prism | dwallet-transfer | stealth-sweep | guest-bind` kinds. Dispatcher is next session — today the envelope is just a shape.

### AggronBatcher DO

Queue + alarm + 4-callable surface. No producers wire to it yet — the sync `POST /api/aggron/publish` endpoint serves today's traffic. DO becomes load-bearing when multi-producer traffic starts.

### SuiInboundWatcher DO

Polls ultron's Sui address every 60s via JSON-RPC `queryTransactionBlocks`, decodes sub-cent intents, logs routing decisions. Forwarding (IKA-signed tx to the real recipient) is stubbed; v4.6 wires that.

### Weavile Surf — Sui stealth math

`deriveSuiStealthForEvent` + `suiAddressFromEd25519Pubkey` + `SUI_SIG_FLAG_ED25519` constants. Consumed by Beldum Double Hit; scanner branch pending.

### Sneasel Slash

Move-side `rotate_sweep_delegate` entry + 4 tests. Client-side `buildRotateSweepDelegateTx` helper. Move upgrade is ready but hasn't been published (plankton holds the UpgradeCap).

### sendWhelm hook

`await sendWhelm('ultron', 5)` → 5 WAL to `sui@ultron` via SUIAMI resolve. Works for any identity-chain combo once the coin type is in the list.
